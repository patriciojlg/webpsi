== Description ==
Elegant Pink is a versatile Masonry based WordPress theme that can be used for building lively and eye-catching websites. The responsive design ensures that your website not only looks great on every device including desktops, laptops, mobiles and tablets, but also provide easy navigation on various types of displays. The theme is loaded with features and tools that allows customization of different aspects of the theme. The theme features multiple post formats, custom menu, custom footer, custom slider and customizer options to create a visually appealing website. The masonry works by placing elements in the optimal position based on the available space, so that your posts are automatically presented in the most visually pleasing and organized way. The masonry  layout makes the theme an ideal option for blogs, portfolios, and other image centric websites. The theme is translation ready. Check out Theme Documentation at http://raratheme.com/documentation/elegant-pink/, Support at http://raratheme.com/support-forum/ and Demo at http://raratheme.com/previews/?theme=elegant-pink.


== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
